**# Random Name Picker**

A simple Python terminal game that randomly selects a winner from a list of names.

**## How it works**
- Enter names separated by commas
- The program splits them into a list
- A random winner is selected and displayed

**## How to run**
```
python main.py
```

**## Built with**
- Python
- `random` module
